== Description ==

Bring holiday cheer to your site with ChristmasPress — a festive, responsive WordPress theme created for seasonal bloggers, events, and joyful celebrations. With a built-in Christmas countdown widget, snowflake-styled sidebars, and nostalgic color schemes, ChristmasPress wraps your content in vintage charm and holiday spirit.

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme via Appearance > Themes
3. Customize as desired under Appearance > Customize
4. Add the Christmas Countdown widget via Appearance > Widgets

== Frequently Asked Questions ==

= Does this theme support the Block Editor? =  
ChristmasPress is a classic-style theme and does not include block-specific templates. However, it is fully compatible with the Classic Editor and standard WordPress content blocks.

= Is the theme translation-ready? =  
Yes! ChristmasPress uses the `christmaspress` text domain throughout for translation.
